rel(relationship) 属性可设置或返回到目标文档的关系。规定了从源文档到目标文档的关系。

rev (reverse) 属性规定了从目标文档到当前文档的反向关系。

  1. alternate - 链接到文档的替代版本（例如打印页、翻译或镜像）。
  2. archives - 链接到文档集或历史数据。
  3. author - 链接到文档的作者。
  4. bookmark - 用作书签的永久 URL。
  5. external - 链接到外部文档。
  6. first - 链接到集合中的首个文档。
  7. help - 链接到帮助文档。
  8. index - 链接到文档的索引。
  9. last - 链接到集合中最后的文档。
  10. licence - 链接到文档的版权信息。
  11. next - 集合中下一个文档。
  12. nofollow - 链接到未认可的文档，比如付费链接。
  13. noreferrer - 规定当用户跟随该超链接时，浏览器不应发送 HTTP referer 头。
  14. prev - 集合中的前一个文档。
  15. search - 链接到文档的搜索工具。
  16. sidebar - 链接到应该在浏览器边栏中显示的文档。
  17. tag - 当前文档标签（关键词）。
  18. up - 提供指向一个文档的链接。该文档提供当前文档的上下文关系。
  19. canonical - 权威,典范 让搜索引擎知道当前网站中的重复或相似网页中，哪一个页面才是站长想让其抓取与收录的
  20. home - 连接到站点的主页
  21. head - 链接到集合中的顶级文档
  22. friend - 赞助 友情链接，底部的 themes by
  23. stylesheet - 文档的外联样式表 link[type="text/css"]
  24. icon shortcut - 快捷方式 小图标 指定标题栏,地址栏,收藏栏小图标 link[type="image/x-icon"]
